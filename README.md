## What it does
- Anony_blog is a website that users can use as an anonymous journal to write their thoughts or feelings. Users are also able to comment and react to other users' posts.
 
## Setup
***
### Installation
- clone repo
- Navigate to the ```anony_blog/client``` folder
  - In the terminal run ```npm install``` to install dependencies 
- Navigate to the ```anony_blog/server``` folder
  - In the terminal run ```npm install``` to install dependencies 
   
### Usage
- Navigate the terminal to the server folder and run ```npm run start``` to start the server. From there you can access the server on ```http://localhost:3000/```
- In another terminal, navigate to the client folder and run ```npm run start``` to start the website. Once running the response in the terminal will give you the address it is available on.
- Lastly in another terminal navigate to the client folder and run ```npm run dev``` to start the bundler

#### API endpoints available
Currently there are 4 unique endpoints
- **'/create'** - *Creates new posts*
- **'/articles'** - *Returns all of the posts*
- **'/updatearticlereaction'** - *Updates a post with a reaction*
- **'/updatearticlecomment'** - *Updates a post with a comment*
