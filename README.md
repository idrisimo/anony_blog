## Setup
***
### Installation
- clone repo
- navigate to the ```anony_blog/client``` folder
  - In the terminal run ```npm install``` to install dependencies 
- navigate to the ```anony_blog/server``` folder
  - In the terminal run ```npm install``` to install dependencies 
  - 
### Usage
- Navigate the terminal to the server file and run ```npm run start``` to start the server. from there youcan access the server on ```http://localhost:3000/```
- In another terminal, go to the client folder and run ```http-server``` to start the website. Once running the response in the terminal will give you the address it is available on.
- Lastly in another terminal navigate to the client folder and run ```npm run dev``` to start the bundler

#### API endpoints available
Currently there are x unique endpoints
- **'/create'** - *Creates new posts*
- **'/googleapi'** - *checks the google api is working*
- **'/getsearch/:text'** - *returns an array of search results*
